# VIRUSxDSCVRY `/virus/vid/`

Video submissions should be placed in this folder using
the naming convention of `/virus/vid/<virus_name>_desc.md`,
which follows the naming convention of your `Virus Report` 
submissions in `/virus/<virus_name.md>`.

Make a copy of `template/vidTemplate.md` file, modify it
accordingly, and rename and submit it here.

Be sure to include your name and virus-name in the submission.
